# INTERCEPTION

A real-time multiplayer game built with Firebase and modern web technologies.

## Features
- **Real-time Multiplayer**: Join lobbies and interact with other players instantly.
- **Persistent Sessions**: Uses LocalStorage to keep you in the game even if you refresh.
- **Shareable Links**: Easily invite friends with direct join links.
- **Spy Aesthetic**: Immersive dark mode design with terminal-style visuals.

## Setup
Just open `index.html` in your browser. No build step required for this version.

## Technologies
- HTML5 / CSS3 (TailwindCSS)
- JavaScript (ES6 Modules)
- Firebase (Firestore, Auth)
